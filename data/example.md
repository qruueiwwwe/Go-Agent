# FileTool 使用指南

## 功能概述

FileTool 是一个智能文件处理工具，支持以下功能：

- 文件内容解析与总结
- 代码分析和优化
- 格式转换

## 支持的文件格式

| 格式 | 支持情况 |
|-----|--------|
| TXT | ✓ |
| Markdown | ✓ |
| JSON | ✓ |
| PDF | ✓ |

## 代码示例

```go
func main() {
    fmt.Println("Hello, FileTool!")
}
```
